ğŸ¨ Customization
Color Scheme
Modify the CSS variables in the <style> section to change the color scheme:

css
/* Primary colors */
color: #00f7ff;     /* Cyan - main text */
color: #ff00cc;     /* Magenta - minute hand */
color: #00ff99;     /* Green - second hand */
Animation Speed
Adjust animation durations in the CSS:

css
animation-duration: 3s; /* Pulse animation */
animation-duration: 15s; /* Floating elements */
Clock Size
Modify these CSS classes to resize the clock:

css
.clock-circle {
    width: 400px;
    height: 400px;
}
